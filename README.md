## 🏗 System Architecture

The application is built on a heavily optimized **Client-Side Rendering (CSR)** architecture powered by React 18 and Vite.

- **Component Design**: Atomic design principles using `shadcn/ui` (Radix Primitives) for accessible, unstyled core components reshaped via Tailwind CSS.
- **State Management**: React Context for lightweight global UI states and modal dialog triggers.
- **Micro-Interactions**: Custom CSS grid backgrounds, synchronized hardware-accelerated pulses, and strict laser-line keyframes (`translateZ(0)`) bypassing heavy JS physics engines for a locked 60fps experience.
- **Execution Deferral**: Mobile CPU blocking is drastically reduced by deferring all below-the-fold component rendering (using `React.lazy()` and `setTimeout` cascades), securing a 95+ Lighthouse Performance Score.
- **Automated Indexing**: Fully integrated with Bing's `IndexNow` API protocol with dynamic XML sitemaps and strict 130-char SEO metadata configurations.

## 🛠 Getting Started

### Prerequisites
- Node.js >= 18.0.0
- npm >= 9.0.0

### Installation

1. **Clone the repository**
    ```bash
    git clone https://github.com/Satya136-dvsn/satya-spark-portfolio.git
    cd satya-spark-portfolio
    ```

2. **Install dependencies**
    ```bash
    npm ci
    ```
    *(Note: usage of `npm ci` ensures exact version matching from `package-lock.json`)*

3. **Start Development Server**
    ```bash
    npm run dev
    ```
    Access the application at `http://localhost:8080`

### Production Build
```bash
npm run build
npm run preview
```
